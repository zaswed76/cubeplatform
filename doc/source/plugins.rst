Plugins
===========

Добавляем плагин.
-----------------

Все плагины находятся в каталоге plugis.
Каталог содержит такие модули:

#. abcPlugin.py
#. manifest.yaml
#. pluginLoader.py
#. __init__.py

Ваш плагин - это питон пакет в котором находятся обязательные четыре файла:

#. __init__.py
#. app.png
#. config.yaml
#. main.py
